# breakpad_dump_syms

## How to use

### Linux
```
dump_syms_linux path_to_binary_with_debug_symbols >path_to_.sym
```

### MacOS
```
dump_syms_mac -g path_to_binary_debug_symbols_.dSYM path_to_binary >path_to_.sym
```

### Windows
```
dump_syms_windows path_to_pdb >path_to_.sym
```
